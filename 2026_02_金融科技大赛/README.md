## 真任务(官方原文)

信贷面签影像合规风控,两步端到端:
1. **影像分类**:从海量多类型影像中**筛出"面签照片"**;
2. **相似度检测**:对面签照片**向量化+相似度比对**,查 **重复使用 / 跨客户套用**(同一张面签照反复提交、或套到不同客户)。

## 技术路线(对齐官方现有基础 = CLIP)

```
多类型影像 ──①分类(CLIP零样本/微调)筛面签──> 面签照片
   ──②度量学习嵌入(CLIP/DINOv2 + ArcFace/对比)──> 判别性向量
   ──③相似度检索(FAISS/余弦)+去重判定──> 同客户重复 / ⚠️跨客户套用
   ──④阈值策略:扫阈值出P/R/F1,给推荐阈值+权衡(官方明确考)
   ──⑤检测汇总视图demo:统计+可疑对并排+关联业务数据
```

## 创新点(对齐真任务)

1. **分类筛选 + 度量学习去重 的端到端流程**(超越官方现有的 CLIP 零样本基线)
2. **难样本判别**:细微差异的"套用"对(裁剪/压缩/轻微编辑后的同一张照)鲁棒识别
3. **阈值自适应策略**:按业务线分层定阈,P/R 权衡可解释(直接对应官方考点)
4. **跨场景复用**:同一套模型适配不同信贷产品线的影像检测

## 关键指标(对齐真任务,替代旧的 AUC/IoU)

- 面签照片**分类准确率/召回**(筛选不漏不错)
- 相似度检测 **Precision-Recall / F1**,以及**最优阈值选取与不同阈值的 P/R 影响分析**
- **跨客户套用**(最严重违规)的检出率
- 可运行性 + 检测汇总视图 demo

## 技术栈

| 组件 | 选型 |
|---|---|
| 分类器 | CLIP 零样本 / 线性探针 / 轻量微调(筛面签) |
| 嵌入主干 | CLIP / DINOv2 / ViT |
| 度量学习 | ArcFace / 对比学习 / 三元组 |
| 检索 | FAISS(IndexFlatIP + L2归一化≈余弦) |
| Demo | Gradio / FastAPI(检测汇总视图) |
| 算力 | 4090 / 4070S |
