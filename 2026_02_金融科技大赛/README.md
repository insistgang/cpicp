## 技术路线

```
        [输入: 票据 / 证照影像]
              │
       ┌──────┴──────┐
       ▼             ▼
   MLLM 上路        细粒度下路
  (Qwen2.5-VL)    (DINOv3 + 篡改感知头)
       │             │
   版式/字段/印章    像素级篡改定位
   语义理解         + 局部特征
       │             │
       └──────┬──────┘
              ▼
        融合决策头
              │
    [相似度, 真伪概率, 热力图, 文字解释]
```

## 创新点

1. MLLM 语义 + 像素细粒度的**双路联合学习**
2. 相似度 + 真伪 + 篡改定位**三任务共享 backbone**
3. **可解释输出**：篡改热力图 + 自然语言解释
4. **AIGC 鲁棒性专项**：Diffusion 时代金融文档篡改基准

## 关键指标

- 真伪二分类 AUC ≥ 0.95
- 篡改区域 IoU ≥ 0.7
- 相似度 Top-1 准确率 ≥ 90%
- 单图推理 ≤ 2 秒
- AIGC 伪造识别率 ≥ 85%
