## 🚀 重点项目展示

### 项目1：灵犀引擎 - 智能金融产品推荐系统

**竞赛**：第四届中国研究生金融科技创新大赛

#### 核心技术
```
┌─────────────────────────────────────────────────────────┐
│                    两步推荐架构                          │
├─────────────────────────────────────────────────────────┤
│                                                          │
│   第一步推荐          反馈            第二步推荐          │
│   基于客户画像   ──────────────→   基于反馈优化          │
│                                                          │
│   准确率: 35%                       准确率: 75%+          │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

#### 技术亮点
| 指标 | 数值 |
|:---|:---:|
| 推荐准确率提升 | 35% → 75%+ |
| 数据规模 | 110万客户 / 33万事件 |
| 响应时间 | < 100ms |
| 并发能力 | 1000+ QPS |

#### 创新点
- 🎯 **两步推荐策略**：初始推荐 + 反馈优化闭环
- ⚖️ **不平衡数据处理**：解决正样本<1%的难题
- 🆕 **新产品冷启动**：基于属性的智能匹配

#### 技术栈
Python + Node.js + scikit-learn + pandas

### 项目2：违章建筑智能检测系统

**竞赛**：第十一届中国研究生智慧城市技术与创意设计大赛

#### 核心技术
```
┌─────────────────────────────────────────────────────────┐
│               改进GoogLeNet + CBAM                       │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  输入影像 → 预处理 → GoogLeNet → CBAM → 检测输出         │
│                                                          │
│  多尺度融合 + 变化检测 + 属性提取                         │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

#### 性能指标
| 指标 | 数值 |
|:---|:---:|
| mAP@0.5 | 92.3% |
| 精度 | 91.8% |
| 召回率 | 93.1% |
| 推理速度 | 45 FPS |

#### 应用场景
- 🏙️ **城市违建监测**：自动发现屋顶加层、临时搭建
- 📊 **变化检测**：多时相影像对比分析
- 🎯 **精准执法**：GPS定位、属性测量

#### 技术栈
PyTorch + OpenCV + GoogLeNet + CBAM
