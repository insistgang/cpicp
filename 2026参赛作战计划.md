## 💡 单兵作战技巧

### 1. 复用策略
```
已有项目 → 提取通用模块 → 新项目复用

例如：
• 灵犀引擎的数据处理 → 复用到智慧城市项目
• 智慧城市的目标检测 → 复用到网络安全项目
• 网络安全的Agent架构 → 复用到双碳项目
```

### 2. AI提示词模板库
```python
# 保存常用提示词
PROMPT_TEMPLATES = {
    "模型训练": "你是一个深度学习专家...",
    "API开发": "你是一个后端工程师...",
    "数据分析": "你是一个数据科学家...",
    "技术方案": "你是一个技术架构师..."
}
```

### 3. 快速Debug技巧
- 把错误信息直接粘贴给AI
- 提供相关代码片段
- 说明预期行为vs实际行为

### 4. 外包质量控制
- 提供详细的交付标准
- 要求阶段性交付
- 技术内容必须自己审核
