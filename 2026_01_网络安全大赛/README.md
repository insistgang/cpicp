## 核心策略：复用 + 升级

**不重写，只加层。** 去年代码 70-100% 可复用，实际增量工作量 1.5-2 周。

- **主推方向 B：自适应防御闭环**
  - 检测到攻击后自动生成防御策略（规则 / 约束 / 拦截器）
  - 反馈到 Agent 系统，真正实现"闭环"
- **主推方向 C：真实部署 Demo**
  - 带 GUI 的 Agent 攻防演示平台
  - 现场演示：红队攻击 → 系统检测 → 自动防御 → 攻击失败
- **备选方向**：A. MCP 协议层扩展 / D. 多 Agent 协同防御 / E. 量化评估 Benchmark
