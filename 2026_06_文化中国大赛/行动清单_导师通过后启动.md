## 导师通过后的优先动作

1. 确定是否继续《颜氏家训》；若换题，只换语料和叙事，不重写工具链。
2. 补充史料来源：
   - 《颜氏家训》版本流变。
   - 家训文献传播史。
   - 家风家教当代转化案例。
3. 把 `src/corpus/yanshi_selected.txt` 从选段扩充为更可靠的校勘材料或节选语料。
4. 重新生成可视化和 PPTX：

```bash
bash src/run_all_selftests.sh
python src/build_pptx.py
```

5. 用 PowerPoint/WPS 打开 `output/作品_初稿.pptx` 做人工美化。
6. 录制 5 分钟以内语音讲解版 PPT。
7. 用 `originality_check.py` 对讲稿和文案做本地相似度自检。
8. 上传百度网盘，保证链接 1 年有效。
9. 走学校资格审核，由院校统一发送材料清单和网盘链接。
