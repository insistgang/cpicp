## 战略定位

**蹭场，走完流程拿证书，不投入真精力。** 文科赛事评的是文化内涵，技术不是优势。

## 作品主题(已对齐真任务)

**一部"日常色彩"典籍的书籍生命史**——以《颜氏家训》/《朱子家训》/蒙书为锚点,做"数字人文作品+视觉作品"组合:版本流变/刊刻传播 → 家庭教化功能(大传统↔小传统互动) → 当代启示,落到"见物见人"。

## 怎么做(人文研究 + 数字人文,非工程系统)

- **重心 = 文化内涵阐释**(占第一权重):史料梳理、书籍生命史叙事、教化功能与当代价值。
- **技术只作展示辅助**(非评分核心):可用 TF-IDF 主题词云/传播时间轴/互文检索等数字人文可视化(复用 02 similarity 思路,纯文本离线即可),做成一个静态可交互网页 + 精良 PPT。
- ⚠️ 不要做"OCR+翻译系统"——那不是赛题2 要的,会偏题。

## 选典籍策略

**优先**：有"日常生活"色彩的文献
- 蒙书：《三字经》《弟子规》
- 家训：《颜氏家训》《朱子家训》
- 乡规民约 / 日用类书

**避开**：太"精英"的（《道德经》《论语》），与"大众日常"主题对不上

## 提交材料(官方原文,两个文件)

- **作品展示 PPT**(≤500M)
- **带语音讲解的 PPT**(≤500M,**播放 ≤5 分钟**)

- ⚠️ 全程匿名:不得出现校名/校徽/姓名/导师/赛事标识(出现一律无效);提交后不可修改
- 流程:官网注册组队 → 培养单位资格审核 → 院校统一寄送(网盘链接+盖章材料清单)到 `wenhuachina2024@163.com`
