## 作品主题

**典籍焕新——基于多模态大模型的古籍智能解读系统**

## 核心功能（3 个）

1. **古籍页面识别**：上传古籍图片 → OCR 出原文（用现成繁体/古文 OCR）
2. **白话翻译**：Qwen-VL 把古文翻译成现代白话
3. **文化背景讲解**：结合上下文，讲历史背景 + 当代启示

## 选典籍策略

**优先**：有"日常生活"色彩的文献
- 蒙书：《三字经》《弟子规》
- 家训：《颜氏家训》《朱子家训》
- 乡规民约 / 日用类书

**避开**：太"精英"的（《道德经》《论语》），与"大众日常"主题对不上
