# 作品大纲 · 《颜氏家训》的书籍生命史(文化中国赛题2)

> 赛题2「典籍与日常」:从书籍史+日常生活史视角,讲一部典籍如何走进百姓日用("见物见人")。
> 形态:**数字人文作品 + 视觉作品**(官方允许);评分第一权重=**文化内涵**,技术只作展示辅助。
> 交付:作品展示PPT + 带语音讲解PPT(≤5min)。全程匿名。

## 一、选题缘起(对应"需求/主题契合")
- 为什么选《颜氏家训》:中国第一部体系化家训,千年间从士族家学走入寻常百姓家——天然的"大传统↔小传统"转化样本。
- 核心追问(扣官方命题):典籍中的教化智慧如何融入日常生活?

## 二、书籍生命史(核心,见物见人)
1. **成书**:颜之推身世与乱世经历 → 写作动机(教子保家)。
2. **版本流变**:抄本→刻本→明清坊刻普及本→近代标点排印本(可做"版本时间轴"可视化)。
3. **传播下沉**:从士族家学 → 蒙学读物/族谱家规引用 → 民间日用(教化功能落地)。
4. **教化主题**:治家/教子/慕贤/勉学/风操——用 `src/text_tools.py` 抽 TF-IDF 主题词做词云。

## 三、大传统↔小传统的互动(对应"文化内涵深挖")
- 经典文本(大传统)如何被节庆、家礼、乡规民约(小传统)吸收转译。
- 用 `src/similarity_search.py` 找《颜氏家训》与后世家训/蒙书的**互文片段**,可视化"知识传承网络"。

## 四、当代启示(对应"时代价值")
- 家风家教的当代回响;典籍数字化活化(数字人文)的路径。
