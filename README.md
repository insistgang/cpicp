## 🚀 快速开始

### 查看项目详情
```bash
# 查看金融科技创新大赛
cd "第四届中国研究生金融科技创新大赛"
start README.md

# 查看智慧城市大赛
cd "第十一届中国研究生智慧城市技术与创意设计大赛"
start README.md
```

### 运行项目演示
```bash
# 金融科技创新大赛
cd "第四届中国研究生金融科技创新大赛/src/backend"
npm install
node server.js

# 然后在浏览器打开前端页面
start ../frontend/smart_recommendation_frontend_fixed.html
```
