# GuardFall Edge 运行说明

## 1 环境

推荐 Python 3.9+。当前 baseline 使用标准库、numpy、PIL/Pillow 和 matplotlib/imageio 可选组件。真实模型和端侧部署阶段再引入 torch、onnxruntime、rknn-toolkit2、NCNN 或海思工具链。

```bash
cd /Users/insistgang/Downloads/cpicp/2026_04_AI创新大赛/src
```

## 2 一键自测

```bash
MPLCONFIGDIR=/private/tmp/ai04_mpl bash run_all_selftests.sh
```

通过后说明跌倒检测主线和历史 AOI 自测均可运行。

## 3 训练 tiny 时序模型

```bash
/usr/bin/python3 train_fall_model.py
```

默认使用合成片段训练,输出:

- `../output/fall_tiny_model.json`
- `../output/fall_tiny_model_report.json`

当前报告显示 tiny head 为 8 维输入、9 个参数,用于验证短时序判别链路。

## 4 跑 synthetic proxy 评测

```bash
/usr/bin/python3 run_fall_pipeline.py \
  --model ../output/fall_tiny_model.json \
  --params-m 0.001
```

输出:

- `../output/fall_pipeline_report.json`
- `../output/fall_scores.npz`

报告字段:

- `metrics.precision_at_recall90`: 召回率 90% 工作点精准率。
- `metrics.precision_at_recall95`: 召回率 95% 工作点精准率。
- `profile.mean_latency_ms_per_frame`: 当前 CPU 后处理单帧平均耗时。
- `competition_score_proxy`: 按赛题评分思路计算的代理分。

注意: synthetic proxy 不是公开视频或官方测试成绩。

## 5 生成可视化

```bash
/usr/bin/python3 viz_fall_report.py \
  --model ../output/fall_tiny_model.json
```

输出:

- `../output/fall_score_distribution.png`: 跌倒/非跌倒分数分布。
- `../output/fall_pr_curve.png`: PR 曲线。
- `../output/fall_demo_contact_sheet.png`: 报警示例拼图。

## 6 公开视频数据评测

下载并解压 OmniFall、Kaggle Fall Video Dataset 后,先生成 manifest:

```bash
/usr/bin/python3 fall_public_datasets.py \
  --omnifall /path/to/omnifall \
  --kaggle /path/to/kaggle_fall_video \
  --out ../output/fall_public_manifest.csv
```

再体检 manifest:

```bash
/usr/bin/python3 fall_data_audit.py \
  --manifest ../output/fall_public_manifest.csv \
  --sample-readable 20 \
  --out ../output/fall_public_audit.json
```

确认 `ready_for_eval` 为 `true` 后再评测:

```bash
/usr/bin/python3 run_fall_pipeline.py \
  --manifest ../output/fall_public_manifest.csv \
  --frames 32 \
  --width 160 \
  --height 120 \
  --skip-errors
```

如需用公开视频重新训练 tiny head:

```bash
/usr/bin/python3 train_fall_model.py \
  --manifest ../output/fall_public_manifest.csv \
  --frames 32 \
  --width 160 \
  --height 120 \
  --skip-errors
```
