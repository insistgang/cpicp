# 跌倒检测赛题提交材料母稿

当前主线: 第八届中国研究生人工智能创新大赛企业赛题七「面向低算力端侧平台基于视觉的实时跌倒检测」。

本目录用于沉淀初赛提交材料的可编辑母稿,暂不直接覆盖根目录下已有 Word 文件。生成正式 PDF/DOCX 前,先在这里确认内容、口径和匿名化风险。

## 文件说明

- `01_参赛作品简介_300字.md`: 300 字以内作品简介,后续导出为 PDF。
- `02_项目文档_初稿.md`: 按附件 2 模板整理的项目文档初稿。
- `03_项目视频脚本_5分钟.md`: 不超过 5 分钟项目视频脚本和镜头清单。
- `04_运行说明.md`: 代码、模型、公开数据评测和产物说明。
- `05_匿名化与提交清单.md`: 初赛材料命名、匿名化和提交前检查。

## 当前口径

- 已实现的是可运行 baseline: 合成片段、纯视觉特征、tiny 时序模型、报警后处理、指标报告和可视化。
- 当前报告中的 P@R90/P@R95 来自 synthetic proxy,只能证明流程闭环,不能写成公开/官方成绩。
- 正式提交前必须接入 OmniFall、Kaggle Fall Video Dataset 或赛题平台参考数据,补充真实视频评测和端侧部署实测。
