## 📊 我的常用配置（保存备用）

### 科研论文风格配置
```python
# 保存为 plot_config.py
import matplotlib.pyplot as plt
import matplotlib

# 字体设置
plt.rcParams['font.sans-serif'] = ['Arial']
plt.rcParams['axes.unicode_minus'] = False

# 配色方案
COLORS = {
    'primary': '#2E86AB',      # 主色：蓝
    'secondary': '#A23B72',    # 辅色：紫红
    'highlight': '#F18F01',    # 高亮：橙
    'success': '#C73E1D',      # 成功：红
    'neutral': '#6B7280'       # 中性：灰
}

# 输出设置
DPI = 300
FIGURE_SIZE = (10, 6)

# 使用示例
# plt.figure(figsize=FIGURE_SIZE)
# plt.bar(..., color=COLORS['primary'])
# plt.savefig('output.png', dpi=DPI)
```

**提示词生成这个配置：**
```
生成一个matplotlib配置文件plot_config.py，
包含科研论文常用的字体、配色、尺寸设置。
配色要专业、适合打印。
```

## 🆘 遇到问题怎么办

### 问题1：AI生成的代码报错
**解决：** 把错误信息复制给AI，说："代码报错：[粘贴错误]，请修复"

### 问题2：图表样式不满意
**解决：** "修改图表：把[某处]改成[某种样式]，重新生成代码"

### 问题3：不会运行Python
**解决：** 
1. 让AI生成"如何在Windows/Mac运行Python"的教程
2. 或使用在线工具：Google Colab（免费，无需安装）

### 问题4：需要更复杂的图
**解决：** 分步骤生成：
1. 先生成基础框架
2. 再添加细节
3. 最后调整样式
