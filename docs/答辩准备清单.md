## 📊 各项目答辩要点

### 1. 金融科技创新大赛 - "灵犀引擎"

**核心亮点**：
- 两步推荐策略：35% → 75%准确率
- 极不平衡样本处理（正样本<1%）
- 110万真实客户数据验证

**答辩重点**：
1. 问题背景：银行精准营销难题
2. 解决方案：两步推荐架构
3. 技术创新：不平衡数据处理
4. 实验结果：准确率提升5倍
5. 应用价值：预计增收数千万元

**可能提问**：
- Q: 如何处理数据隐私？
  A: 数据脱敏，标识匿名化
- Q: 两步推荐的时间成本？
  A: 每步<100ms，总时间<200ms
- Q: 如何防止推荐疲劳？
  A: 推荐间隔控制，多样化策略

## 🎤 演讲技巧

### 开场（30秒）
```
各位专家好！我是来自上海工程技术大学的刘钢。
今天我答辩的项目是【项目名称】，
我们的核心理念是【一句话概括】，
创新点是【关键技术】，
下面我将从【几个方面】进行介绍。
```

### 主体（13分钟）
- 每页PPT控制在1分钟左右
- 重点页面可延长至2分钟
- 保持语速适中，声音洪亮
- 与评委保持眼神交流

### 结尾（1分钟）
```
总结一下，我们的项目【核心成果】，
具有【应用价值】，
感谢各位专家的聆听，
欢迎批评指正！
```

## ❓ 常见问题准备

### 技术类问题
1. **Q**: 你们的方法与现有方法相比有什么优势？
   **A**: 从准确率、效率、成本等角度对比

2. **Q**: 实验数据是否可靠？
   **A**: 说明数据来源、规模、验证方法

3. **Q**: 系统的可扩展性如何？
   **A**: 架构设计支持水平扩展

### 应用类问题
1. **Q**: 这个项目有什么实际应用价值？
   **A**: 量化经济效益和社会效益

2. **Q**: 是否已经落地应用？
   **A**: 说明当前状态和后续计划

3. **Q**: 商业模式是什么？
   **A**: 面向B端/G端的商业模式

### 团队类问题
1. **Q**: 你在团队中负责什么？
   **A**: 明确分工，突出个人贡献

2. **Q**: 遇到的最大困难是什么？
   **A**: 技术难点及解决方案
