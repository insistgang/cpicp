## 申请流程

### 第一步：准备材料（1-2周）

#### 1.1 源代码
- 要求：前30页 + 后30页，共60页
- 格式：A4纸，小四号字，单倍行距
- 每页50行以上
- 页眉需包含软件名称和版本号

#### 1.2 文档材料
- **用户手册** 或 **设计说明书**
- 15页以上，需图文结合
- 包含软件功能、操作流程等

#### 1.3 申请表
- 在中国版权保护中心官网填写
- 需要学校盖章
