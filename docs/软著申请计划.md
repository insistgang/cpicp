## 📄 申请材料清单

每个软著申请需要准备以下材料：

### 1. 源代码（必须）
- **要求**：前30页 + 后30页 = 共60页
- **格式**：A4纸，小四号字，单倍行距
- **行数**：每页50行以上
- **页眉**：需包含软件名称和版本号

### 2. 文档材料（必须）
- **用户手册** 或 **设计说明书**
- **页数**：15页以上
- **内容**：图文结合，包含功能说明、操作流程等

### 3. 申请表（必须）
- 在中国版权保护中心官网在线填写
- 需要学校盖章
- 需要指导教师签字

### 4. 身份证明（必须）
- 申请人身份证复印件

## 🏫 学校申请流程

### 步骤1：准备材料（个人）
1. 整理源代码60页
2. 撰写用户手册15页+
3. 填写申请表

### 步骤2：学校审核
1. 提交材料至研究生处/科技处
2. 等待审核盖章
3. 获取学校推荐意见

### 步骤3：在线提交
1. 登录中国版权保护中心官网
2. 注册账号并填写申请表
3. 上传材料PDF
4. 邮寄纸质材料

### 步骤4：等待审批
1. 形式审查（1个月）
2. 实质审查（1个月）
3. 发证（2周）

**总耗时**：约3个月
