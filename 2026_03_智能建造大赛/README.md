## 技术路线

```
SeaDronesSee 数据集
        ↓
    YOLOv12-n/s 训练(4090)
        ↓
   ┌────┴────┐
   ▼         ▼
小目标增强   轻量化
- BiFPN/PANet 多尺度融合
- CBAM/EMA 注意力机制
- 小目标 anchor 优化
- 水面噪声数据增强
        ↓
   ONNX → TensorRT
        ↓
Jetson Orin Nano 8G 部署
INT8 量化 + DeepStream 加速
        ↓
   ≥ 30 FPS + mAP 报告
        ↓
(可选) QGroundControl demo
```

## 创新点

1. 针对水面**小目标**的多尺度特征融合改进
2. 水面**光照反射/波纹**的数据增强策略
3. **Jetson Orin Nano 上的 TensorRT-INT8 加速管线**

## 关键指标

- 动态视频 ≥ 30 FPS（**必须真实硬件实测**）
- 各类目标（落水人员/船只/浮标）Precision-Recall 曲线 + mAP

## 提交材料

1. 技术方案文档（YOLOv12 选型 + 小目标增强 + 轻量化思路）
2. 算法源代码（Python，可复现）
3. 性能评估报告（PR 曲线 + Jetson 实测 FPS）
4. 应用演示（QGroundControl 视频流融合，加分项）

## 关键风险

- ⚠️ **30 FPS 是硬卡线**，达不到直接出局
- ⚠️ Jetson Orin Nano 8G 显存：YOLOv12-s 可行，YOLOv12-m 需量化
